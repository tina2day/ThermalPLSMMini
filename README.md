# Thermal Video Action Recognition

Классификация действий человека по тепловым видеоданным.

## Описание

Проект содержит pipeline для обработки тепловых видео, формирования событийных признаков и обучения модели классификации действий.

## Структура

- `thermal_action_recognition_plsm.ipynb` — основной ноутбук эксперимента;
- `data/` — папка с датасетом;
- `checkpoints/` — сохранённые веса модели;
- `training_plots/` — графики обучения.
